Метою роботи є побудова теорії ефективних електричних властивостей невпорядкованих дисперсних систем частинок з морфологією тверде ядро--проникна оболонка. Оболонки в загальному випадку мають неоднорідний радіально-симетричний профіль електричної провідності. Правила домінування, що накладаються на області перекриття оболонок, еквівалентні вимозі, що локальне значення провідності визначається відстанню до центру найближчої до даної точки частинки. Для побудови теорії узагальнюється на провідні системи та застосовується метод компактних груп, що дозволяє врахувати багаточастинкові поляризаційні і кореляційні ефекти без їх деталізації. Теорія замикається доповненням її вимогою виконання граничних умов для нормальних компонент густини комплексного струму. Результуюче значення ефективної квазістатичної комплексної проникності залежить від комплексних проникностей компонент системи та їх об’ємних концентрацій, що визначаються статистичною мікроструктурою системи. У статичному наближенні теорія є строгою.

Тестування теорії виконується шляхом порівняння її результатів з широким масивом даних числових симуляцій, отриманих методом Random Resistor Network, для залежностей об’ємної концентрації оболонок та статичної провiдностi від концентрації твердих ядер розглядуваної модельної системи у випадку, коли провідності ядер та матриці набагато менші за провідність оболонки, який є характерним для твердих композитних (ТКЕ) та полімерних композитних (ПКЕ) електролiтiв. Числові дані отримані для різних дiаметрiв ядер та товщин оболонок двох типів: електрично однорідних та електрично неоднорідних з гаусовим радіальним профілем провiдностi. Для врахування виявленої в роботі неточності відображення симуляціями об’ємної концентрації оболонок вводиться підгінний параметр, що дозволяє відновити всі числові дані для першого типу оболонок; для відновлення даних для другого типу вводиться ще один підгінний параметр, який враховує ефективну зміну висоти гаусового профілю провiдностi сферично-симетричних оболонок алгоритмом.

Після цього теорія використовується для аналізу експериментальних даних для квазістатичної електричної провідності твердих композитних та полімерних композитних електролітів. Отримані результати обробок цих даних свідчать про те, що в формуванні ефективної провідності на різних концентраційних інтервалах беруть участь різні механізми, які в рамках теорії ефективно враховуються профілем провідності оболонок. Це додатково підтверджується результатами обробки температурних залежностей, використовуючи емпіричний закон Фогеля-Таммана-Фульхера для врахування температурних залежностей провідностей аморфізованих областей оболонок та матриці.

Далі аналізується ефект перколяції електричної провідності та діелектричної проникності в системах типу провідник--ізолятор з електрично однорідним міжфазним шаром. Положення порогу перколяції визначається як концентрація при якій виникає ненульова ефективна статична провідність в системі з непровідною матрицею. В моделі поріг перколяції визначається відносною товщиною оболонки, а значення ефективних критичних індексів для цих систем залежать як від геометричних та електричних параметрів компонентів, так і способу обробки експериментальних даних, а тому демонструють широкий спектр значень спостережуваних на експерименті. При обробці експериментальних даних з ефективних квазістатичних діелектричної проникності та електричної провідності систем на основі KCl з частинками Ag з проникними оксидними шарами теорія демонструє себе як ефективний інструмент електроспектроскопічного аналізу та дозволяє кількісно оцінити товщину оксидної оболонки.

Також в рамках розвиненого формалізму показується, що диференціальна схема аналізу ефективних квазістатичних електричних параметрів дисперсних систем є застосовною лише для систем з малими різницями діелектричних проникностей компонентів у вузьких концентраційних інтервалах диспергованих компонентів.
