\vskip 15pt

В роботі побудовано модель квазістатичного електричного відгуку невпорядкованих тривимірних систем частинок з морфологією тверде ядро--проникна оболонка. Оболонки в загальному випадку електрично неоднорідні. Локальне значення провідності в системі визначається одночастинковим профілем провідності найближчої частинки. Для побудови теорії використовується метод компактних груп, що дозволяє врахувати багаточастинкові поляризаційні і кореляційні ефекти без їх деталізації. Теорія замикається доповненням її вимогою виконання граничних умов для нормальних компонент густини комплексного струму.

Модель тестувалася шляхом порівняння її результатів з широким масивом даних числових симуляцій, отриманих методом Random Resistor Network для залежностей  об’ємної концентрації оболонок та статичної провідності від концентрації твердих ядер для різних діаметрів ядер та товщин електрично однорідних та неоднорідних оболонок. Показано, що всі результати можна відновити беручи до уваги неточності відображення модельної системи розглядуваним алгоритмом за рахунок введення одного підгінного параметру для електрично однорідних оболонок та двох -- для електрично неоднорідних.

Показано застосовність теорії до кількісного опису експериментальних даних з концентраційних залежностей ефективної провідності твердих композитних та полімерних композитних електролітів та аналізу ролі різних фізико-хімічних механізмів у її формуванні. Внески останніх можна ефективно врахувати через модельний профіль комплексної діелектричної проникності проникних оболонок, що підтверджується застосуванням теорії до опису температурних залежностей провідності. Також показано, що теорія спроможна кількісно описувати експериментальні дані з ефективних електричної провідності та діелектричної проникності систем типу ізолятор--провідник з міжфазним проникним шаром в околі порогу електричної перколяції. Останній в рамках теорії залежить лише від відносної товщини оболонки; ефективні критичні індекси провідності таких систем залежать лише від геометричних та електричних параметрів компонентів та способу обробки експериментальних даних. 

Також показано, що диференціальна схема аналізу ефективних квазістатичних електричних параметрів дисперсних систем є застосовною лише для систем з малими різницями діелектричних проникностей компонентів у вузьких концентраційних інтервалах.
